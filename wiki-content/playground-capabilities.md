# AI Chatbot Capabilities - Hướng dẫn sử dụng

## Chatbot có thể làm gì?

### 1. Trả lời câu hỏi
Hỏi bất cứ thứ gì — khoa học, lịch sử, địa lý, đời sống.
> Ví dụ: "Vì sao bầu trời màu xanh?"

### 2. Viết code
Hỗ trợ Python, JavaScript, Java, C++ và nhiều ngôn ngữ khác.
> Ví dụ: "Viết hàm Python kiểm tra số nguyên tố"

### 3. Dịch thuật
Dịch giữa tiếng Việt và nhiều ngôn ngữ.
> Ví dụ: "Dịch câu này sang tiếng Anh: Tôi yêu Việt Nam"

### 4. Tóm tắt văn bản
Gửi văn bản dài, chatbot sẽ tóm tắt ý chính.
> Ví dụ: "Tóm tắt bài viết này trong 3 câu"

### 5. Viết sáng tạo
Viết thơ, truyện, bài blog, email chuyên nghiệp.
> Ví dụ: "Viết một bài thơ về mùa xuân Hà Nội"

### 6. Toán học
Giải phương trình, tính toán, giải thích công thức.
> Ví dụ: "Giải phương trình x² + 5x + 6 = 0"

### 7. Phân tích dữ liệu
Phân tích, so sánh, đánh giá thông tin.
> Ví dụ: "So sánh ưu nhược điểm của React và Vue.js"

### 8. Tư vấn
Đưa ra gợi ý, lời khuyên về nhiều chủ đề.
> Ví dụ: "Cho tôi lời khuyên để học tiếng Anh hiệu quả"

## Mẹo để có kết quả tốt nhất

1. **Câu hỏi cụ thể**: Thay vì "Hướng dẫn code", hãy hỏi "Viết Python code đọc file CSV và tính trung bình cột Age"
2. **Cung cấp ngữ cảnh**: "Tôi là sinh viên năm nhất, giải thích đơn giản về..."
3. **Yêu cầu format**: "Liệt kê 5 điểm", "Dạng bảng", "Ngắn gọn trong 2 câu"
4. **Hỏi tiếp**: Nếu chưa rõ, hãy hỏi lại hoặc yêu cầu giải thích chi tiết hơn

## Tính năng đặc biệt

- 🔍 **Tìm kiếm web**: Gõ `/search:` trước câu hỏi để tìm thông tin mới nhất
- 📎 **Gửi ảnh**: Chế độ Lite hỗ trợ phân tích hình ảnh
- 🧹 **Xóa lịch sử**: Gõ `/clear` để bắt đầu cuộc trò chuyện mới
- ⚡ **Streaming**: Bật/tắt chế độ trả lời realtime
