# Chatbot Knowledge: product_info

Dùng domain này cho thông tin sản phẩm/dịch vụ.

Ví dụ card nên có:

- Mô tả từng sản phẩm/gói dịch vụ
- Tính năng chính
- Đối tượng phù hợp
- Giới hạn sử dụng
- So sánh giữa các gói

Không đặt ở đây:

- Giá/khuyến mãi/hoàn tiền → `pricing_policy`
- Điều khoản pháp lý/bảo hành → `terms_policy`
- Cách xử lý lỗi → `troubleshooting`
